leetcode 88. 合并两个有序数组

2020-10-23 leetcode

思路

1.首先能想到，类似归并排序中的归并操作。

2.但是这里有一点难度在于，我们要复用nums1这个数组并达到O(1)的空间复杂度。

3.一个有序数组，可以是非严格的（比如 1， 2， 2，3）升序，也可以是降序。

我们先来思考降序的，我们可以先将nums1中的非零元素移到尾端 在归并过程中无论如何 都需要额外的空间来保存元素

所以降序肯定不行

此时 我发现题目是升序的 囧 那就逆序归并就好了

func merge(nums1 []int, m int, nums2 []int, n int)  {
    i := m + n - 1
    index1, index2 := m - 1, n - 1
    for index2 >= 0 && index1 >= 0 {
        if (nums1[index1] > nums2[index2]) {
            nums1[i] = nums1[index1]
            index1--
        } else {
            nums1[i] = nums2[index2]
            index2--
        }
        i--
    }
    for index2 >= 0 {
        nums1[i] = nums2[index2]
        i--
        index2--
    }
}
