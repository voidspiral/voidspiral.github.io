排序数组，平方后，数组当中有多少不同的数字

2020-10-27 2 min read

题目

排序数组，平方后，数组当中有多少不同的数字（相同算一个）

思路

  1. 初始化两个左右坐标，指向数组的首尾，向中间遍历，初始化 pre 变量为 abs(nums[0]-1) 表示上一次记录过的数字， res=0
  2. 左边比右边的绝对值大时，左边向右遍历，若左侧数字没出现过，则 res++，pre = 左侧数字
  3. 右边比左边的绝对值大或等于时，右边向左遍历，若右侧数字没出现过，则 res++，pre = 右侧数字
package main

import (
   "fmt"
   "math"
)

func solution(nums []int) int {
   if len(nums) < 1 {
      return 0
   }
   left, right := 0, len(nums)-1
   pre := int(math.Abs(float64(nums[0] - 1)))
   res := 0
   for left < right {
      if nums[left] > nums[right] {
         if nums[left] != pre {
            res++
            pre = nums[left]
         }
         left++
      } else {
         if nums[right] != pre {
            res++
            pre = nums[right]
         }
         right--
      }
   }
   return res
}
func main() {

   nums := []int{-1, -2, 0, 2, 3, 4}
   fmt.Println(solution(nums))
}
